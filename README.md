# Gradio Groq Chatbot

A customizable AI chatbot built using **Gradio** and **Groq**.

## ✨ Features

- Custom System Prompt
- Temperature Control
- Real-Time Streaming Responses
- Full Chat History
- Powered by Groq Llama Models
