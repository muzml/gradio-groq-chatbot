# Gradio Groq Chatbot

A customizable AI chatbot built with Gradio and Groq that supports:

- Custom System Prompts
- Temperature Control
- Real-Time Streaming Responses
- Chat History Memory
- Llama 3.3 70B via Groq

## Installation

pip install -r requirements.txt

## Run

python app.py

## Tech Stack

- Python
- Gradio
- Groq API
- Llama 3.3 70B
